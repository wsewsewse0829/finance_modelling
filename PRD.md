# 个人财务规划应用 PRD

## 产品概述
线上化的轻量个人财务规划应用，采用企业财务管理的思维管理个人财务，帮助自己做好个人的财务规划，也可以用该应用做上市公司财报的简单分析。

## 目标用户
有一定的财务管理知识，不需要时刻记账，但需要至少每月review个人财务状况，做好财务规划的个人用户

## 核心功能

### 必须做（MVP）
1. 创建序时帐/科目余额表/会计报表/科目表
2. 实现用户自定义管理科目表，批量上传序时帐，应用自动生成科目余额表和会计报表
3. 系统自动校验序时帐借贷是否平衡，会计报表是否平衡
4. 会计报表可视化呈现

### 后续可以做
1.基础功能迭代
   - 数据导出
   - 自定义可视化图表
   - 用户账户注册，多账户管理
2. 滚动测试功能
   - 用户自定义预测逻辑输出未来期间的会计数据，与实际数据一起在会计报表可视化页面呈现
   - 用户设定参数，实现调整参数估计，改变预测结果并能可视化呈现
   - 自定义可视化图表
   - 用户账户注册，多账户管理
3. 科目明细表创建
   - 用户自定义科目明细数据：如分期付款数据、资产摊销数据、投资组合数据等
   - 个人投资组合管理表：包括本金、投资组合成分标的、盈利情况以及夏普比率
   - 科目明细表可以和会计报表实现联动
4. 财务比率分析
   - 根据分析用途自定义管理报表口径
   - 杜邦分析
   - 市盈率、市净率等相对指标分析
   - 可比公司对比分析
5. 个人轻博客
   - 用于记录各类事项
   - 可以设置评论区讨论

## 界面设计
- 首页：页面切换栏、主要功能介绍、用户注册（用户注册功能待后续开发）
- 页面1：会计报表（目前只需要会计报表展示：每行为会计科目，每列为每月/每年度，报表顺序为：资产负债表、利润表、现金流量表、所有者权益变动表，后续跟进开发进度添加参数调整按钮、图表、比率分析等）
- 页面2: 科目管理：用户自定义次级科目
- 页面3：数据上传：用户上传会计数据
- 其他页面待后续开发
